# Data Directory

Dataset files are **not** tracked in this repository due to size and licensing. Place downloaded files in the paths below before running the notebooks.

See the [Datasets & Sources](../README.md#datasets--sources) section in the main README for download links and setup instructions.

| Path | Used by |
|------|---------|
| `adult/adult.data`, `adult/adult.test` | `notebooks/subpopulation_attack/subpopulation_attack_paperimplementation.ipynb` |
| `stroke/healthcare.csv` | `notebooks/subpopulation_attack/Subpopulation_Stroke.ipynb` |
| `dogvscat_traindata/` | `notebooks/trash_bucket/dogvscat.ipynb` |
| `dogvscat/` (saved models + `test/` images) | `notebooks/trash_bucket/dogvscat.ipynb` |
| `omniglot/` | `notebooks/one_shot_learning/one_shot_learning_omniglot.ipynb` |

The file `adult/adult.names` (dataset description) is kept in the repo for reference.
